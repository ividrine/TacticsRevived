# TacticsRevived

Live service, turn based strategy game inspired by *Tactics Arena: Online*, a web based flash game from the early 2000s.

![In-Game movement example](movement-example.gif)

## Features
- Game portal with UI classes to support authentication, chat and 1v1 matchmaking
- Base game with custom board logic and unit classes
- Player controls for moving units on the game board

## Roadmap
- Gameplay mechanics (attacking, abilities, turn order)
- Unit board layout management
- Additional game modes (PvE)
- Collections (skins, icons, etc)
- Store and in-game currency
